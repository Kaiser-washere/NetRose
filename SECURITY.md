# Security Policy

## Reporting a Vulnerability
- Report vulnerabilities privately via Discord: **kaiserwashere**.
- Do not disclose publicly until a fix has been released.

## Disclosure Policy
- Vulnerabilities must remain private until resolved.
- Public disclosure before a fix is prohibited.

## Supported Versions
- Security updates are provided only for the **latest release** of NetRose.

## Response Time
- Reports will be acknowledged and addressed within a **reasonable timeframe**.
