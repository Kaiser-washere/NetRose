# Contributing to NetRose

We welcome contributions to improve NetRose. Please follow these guidelines:

## Process
- Fork the repository and create a new branch.
- Make your changes and write clear commit messages.
- Open a Pull Request (PR) describing what you changed.

## Code and Documentation
- Write clean, readable code.
- Add tests where possible.
- Documentation updates and usage examples are encouraged.

## License

## Communication
- For questions or discussions, join our Discord: [discord.com]
